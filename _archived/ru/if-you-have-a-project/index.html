Если у вас есть задача/проект или вы заинтересованы в работе со мной, я с радостью открыт для сотрудничества по следующим направлениям:

  • Консультирование / разработка ПО. В области алго-трейдинга, моделирования, архитектуры ПО, фреймворков.
  • Риск-менеджмент проекты / консультирование. Оценка рисков портфелей, включая сложные позиции на опционах, фьючерсах. Хеджирование валютных, процентных и товарных рисков.
  • Разработка торговых систем. Все что связано с автоматизацией трейдинга: бэктестинг, оптимизация, исполнение ордеров, дашборды, роботы.
  • Торговля. Если есть свободный капитал, у меня есть идеи как его эффективно разместить.
  • Исследования рынков. Помогу найти профит или идеи для FinTech продуктов.

Условия и ограничения

Удаленная работа, без переезда, частичная занятость (проектный подход). Предпочитаю среднесрочные / долгосрочные проекты, длительностью от 3-х месяцев. Обычно это проекты с фиксированной ценой, разбитые на этапы по 2-4 недели каждый. Для каждого проекта пишу спецификацию, где отмечаются четкие критерии успеха и шаги по реализации.

Беру проекты долгосрочной стоимостью от $5000, с полноценной занятостью 10-20 часов/неделя, когда проект в активной фазе разработки. Обычно с клиентом начинаем работать с небольшого концепта, и поэтапно развиваем до полноценного продукта.

Работаю как ИП, по договору, с клиентами из РФ в рублях, из других юрисдикций в валюте.
